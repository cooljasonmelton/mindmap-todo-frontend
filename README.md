## Application Features

- Create, Read, Update, Destroy todo tasks
- Select task and get breakdown into steps
- Select task and get brainstorming questions about task
- Ask LLM a custom question

## Built With​

front end:

- TypeScript
- React
- Next.js
- Tailwind
- React Query

api:

- Node.js
- Ollama (local)
